API Index

This is the API webpage for testing development.

If you somehow stumbled onto this page, kudos to you, but there really isn't anything special here

There are two functioning api endpoints:

api.ciphercrypt.network/encrypt

Encrypt accepts the following body: 


        type RequestData struct {
            Key  string         `json:"key"`
            Data []PasswordData `json:"data"`
        }
        
        type PasswordData struct {
            Password string `json:"password"`
            Username string `json:"username"`
            Notes    string `json:"notes"`
        }
        
api.ciphercrypt.network/decrypt

Decrypt accepts the following body: 


        type DecryptData struct {
            Key string `json:"key"`
            Salt string `json:"salt"`
            Ciphertext string `json:"ciphertext"`
        }
